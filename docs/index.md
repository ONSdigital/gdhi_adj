This site contains the project documentation for `gdhi-adj`, a project  that
flags outliers of GDHI data at LSOA levels and adjusts the outlier values.

Quickly find what you're looking for depending on your use case by

## Prerequisites

The following prerequisites are required for `gdhi-adj`:

- Python 3.12 or higher
